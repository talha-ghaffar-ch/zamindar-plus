# Zamindar Plus Mobile

React Native CLI app for the Zamindar Plus farm ledger platform.

## Run Locally

From the repository root:

```sh
npm install
npm run dev:api
npm run android -w apps/mobile
```

The Android emulator uses `http://10.0.2.2:3000` to reach the local NestJS API.

## Main Areas

- Secure sign in, sign up, email verification, and password reset flows.
- Mobile dashboard for profit, records, monthly movement, and quick actions.
- Quick add forms for profiles, zameen, crops, expenses, and income.
- Records, reports, settings, and session-only Zamindar AI chat.
